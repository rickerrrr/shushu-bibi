GitHub Token 配置教程

🍾 配置后，你们的漂流瓶可以互相看到啦！
👫 不管用哪台电脑，都能收到对方的瓶子～

💡 重要提示

她不需要自己建GitHub账号！

你生成Token后发给她，她用同一个Token就行～

🔑 使用方法：

1. 你生成Token（ghp_开头那串）
2. 把Token发给她（微信/QQ都行）
3. 她打开网站，粘贴Token，保存
4. ✅ 完成！两人都能看到对方的瓶子了！

🔒 安全说明：Token只有"gist"权限，不能访问其他数据，发给她完全没问题！
1
🌐 打开GitHub Token设置页面

用手机或电脑浏览器打开这个链接：

https://github.com/settings/tokens

💡 如果没登录，会先让你登录GitHub账号

打开GitHub Settings页面
📸 这里可以放截图
2
🎯 点击"Generate new token"

点击 "Generate new token" 按钮

然后选择 "Generate new token (classic)"

点击 Generate new token 按钮
📸 这里可以放截图
3
📝 填写Token信息（照着填！）
  • 📝 Note（备注）： 随便写，比如 shushu-bibi漂流瓶
  • 📅 Expiration（过期时间）： 选择 No expiration（永不过期）
  • ✅ Select scopes（权限）： 只勾选 gist ✅（其他不要勾！）
一定要勾选 gist！不勾的话Token不能用！
勾选 gist 权限
📸 这里可以放截图
4
🚀 点击"Generate token"生成

拉到页面最下面，点击 绿色 的 "Generate token" 按钮

点击绿色 Generate token 按钮
📸 这里可以放截图
5
📋 复制Token（重要！！！）

✅ 生成成功后，会看到一串以 ghp_ 开头的字符串

ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
立即复制！这个页面只显示一次！
关闭后就看不到了！如果没复制，只能重新生成一个！

📋 点击复制按钮，或者手动选中全部文字复制

6
💕 在网站配置Token
  • 📱 打开 shushu-bibi.cn
  • 🔑 登录后，进入 🍾 漂流瓶 模块
  • ⚠️ 会看到黄色提示："未配置 GitHub Token"
  • 👆 点击 "立即配置" 按钮
  • 📋 粘贴刚才复制的 ghp_... Token
  • 💾 点击 "保存"
配置成功！黄色提示会消失，现在可以投瓶子了！

🎉 配置完成！

  • ✅ 现在你可以投瓶子了
  • ✅ 你投的瓶子，男朋友能捞到
  • ✅ 男朋友投的瓶子，你能捞到
  • ✅ 你们的漂流瓶是实时同步的！

  • 1️⃣ 检查Token是否以 ghp_ 开头
  • 2️⃣ 检查生成Token时是否勾选了 gist
  • 3️⃣ 截图发给我，我帮你看看
